Image Lesson

Steps in building an Image

  1. Pick an Image (careful of the file extension type)
  2. Record URL, cite it, comment for date access, and record orginal dimensions
  3. Use blockquote and comment as appropriate
  4. Use img-tag with attributes src, alt, title, and style with width-height
